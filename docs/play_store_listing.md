# Play Store 등록 문구 초안

## 앱 이름

3분세끼

## 짧은 설명

결정하기 귀찮은 한 끼를 3분 안에 골라주는 로컬 메뉴 추천 앱

## 긴 설명

오늘 뭐 먹을지 고르기 어렵다면 3분세끼가 빠르게 후보를 좁혀드립니다.

가격대와 음식 종류를 고른 뒤, 카드처럼 메뉴를 넘기며 좋아요와 싫어요를 선택해보세요. 좋아요를 받은 메뉴는 토너먼트로 한 번 더 비교해 오늘의 한 끼를 결정합니다.

가격만 고르고 바로 뽑는 랜덤 추천도 지원합니다. 못먹는 음식 태그를 설정하면 해당 재료가 포함된 메뉴를 추천에서 제외합니다.

주요 기능:

- 가격대 기반 메뉴 추천
- 한식, 중식, 양식, 일식, 기타 카테고리
- 밥, 면, 육류, 기타 형태 선택
- 카드 스와이프 방식의 빠른 취향 선택
- 좋아요 메뉴 토너먼트
- 가격만 고르는 순수 랜덤 추천
- 못먹는 음식 태그 필터
- 최근 식사 기록 저장과 수정
- 서버 없는 로컬 동작

3분세끼는 로그인 없이 사용할 수 있으며, 추천 설정과 기록은 사용자의 기기 안에 저장됩니다.

## 키워드 참고

메뉴 추천, 음식 추천, 점심 추천, 저녁 추천, 랜덤 메뉴, 오늘 뭐 먹지, 한식 추천, 점심 메뉴

## 스크린샷 제안

1. 온보딩 화면
2. 못먹는 음식 태그 설정 화면
3. 가격 선택 화면
4. 스와이프 카드 추천 화면
5. 토너먼트 화면
6. 랜덤 추천 화면
7. 식사 기록 화면

## 콘텐츠 등급 메모

음식 추천 앱으로 폭력, 성적 콘텐츠, 도박 요소 없음.

## Data Safety 메모

현재 버전 기준:

- 로그인 없음
- 서버 전송 없음
- 위치 정보 수집 없음
- 연락처/사진/파일 접근 없음
- 기기 내 SharedPreferences에 설정과 식사 기록 저장

실제 광고 SDK를 연동하기 전까지는 광고 식별자 수집이 없습니다.
